Este es un proyecto para analizar los sentimientos de las personas en relación a una series de peliculas españolas. Para este proyecto se obtuvo un dataset de Kaggle de la siguiente dirección https://www.kaggle.com/datasets/ricardomoya/criticas-peliculas-filmaffinity-en-espaniol?select=reviews_filmaffinity.csv. La idea es realizar el análisis con un enfóque de procesamiento de lenguaje natural. Observar los comentarios realizados por las personas para crear un modelo que distinga comentarios negativos y positivos y de ser posible precisar una relación entre el tipo de comentario y la propensión a ver las peliculas.
